# NITOR Website - Modern Redesign

## 🏥 About NITOR
National Institute of Traumatology & Orthopaedic Rehabilitation (NITOR) is a leading healthcare institution in Dhaka, Bangladesh, providing comprehensive trauma care and orthopedic rehabilitation services.

## 📋 Project Overview

This is a complete redesign of the NITOR hospital website, modernizing it with:
- Professional, responsive web design
- Improved user experience (UX)
- Enhanced functionality and features
- Better content organization
- Mobile-first approach
- Accessibility support

## 🎬 Getting Started

### **To View the Website:**

1. Open `index.html` in a modern web browser
2. Or deploy to a web server (GitHub Pages, Netlify, Vercel, etc.)

### **To Customize:**

1. **Change Colors**: Edit CSS variables in `styles.css`
   ```css
   :root {
       --primary-color: #0066cc;
       --secondary-color: #00a86b;
       /* ... */
   }
   ```

2. **Update Content**: Edit text, images, and information in `index.html`

3. **Add Features**: Extend `script.js` with additional JavaScript functionality

4. **Add Images**: Replace placeholder icons with actual images
   ```html
   <img src="path/to/image.jpg" alt="Description">
   ```
